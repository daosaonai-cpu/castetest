C Programming Loops & Control Statements

A simple guide with easy-to-understand examples.

For Loop

The `for` loop is used to execute a block of code a specific number of times. It's great when you know exactly how many times you want to loop.

// Prints numbers from 1 to 5
#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

While Loop

The `while` loop executes a block of code as long as a specified condition is true. It checks the condition *before* executing the loop's body.

// Prints numbers from 1 to 5
#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

Do-While Loop

Similar to a `while` loop, but it executes the block of code *at least once* before checking if the condition is true.

// Prints numbers from 1 to 5
#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d\n", i);
        i++;
    } while (i <= 5);
    return 0;
}

Break Statement

The `break` statement is used to exit a loop immediately, regardless of the loop's condition. Execution continues at the first statement after the loop.

// Stops the loop when i is 3
#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 5; i++) {
        if (i == 3) {
            break; // Exit loop
        }
        printf("%d\n", i);
    }
    // Output: 1 2
    return 0;
}

Continue Statement

The `continue` statement skips the current iteration of a loop and moves to the next one. The loop's condition is re-evaluated.

// Skips printing 3
#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 5; i++) {
        if (i == 3) {
            continue; // Skip this iteration
        }
        printf("%d\n", i);
    }
    // Output: 1 2 4 5
    return 0;
}

Quick Summary

  • for: Use when you know the number of iterations.
  • while: Use when the number of iterations is unknown and depends on a condition.
  • do-while: Use when you need the loop body to execute at least once.
  • break: To exit a loop early.
  • continue: To skip the current iteration and proceed to the next.
