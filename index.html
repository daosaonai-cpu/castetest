Interactive C Loops Tutorial

Master `for`, `while`, and `do-while` loops in C with hands-on examples.

The `for` Loop

The `for` loop is ideal when you know in advance how many times you want to execute a block of code. It combines initialization, condition checking, and an increment/decrement operation in a single line, making the code compact and readable.

Syntax:

for (initialization; condition; update) {
    // code to be executed
}

Example 1: Print Numbers 1 to 5

A classic example that prints numbers from 1 to 5, demonstrating the basic structure.

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n");
    return 0;
}

Example 2: Print Multiplication Table of 7

This code generates the multiplication table for a given number (7 in this case) up to 10.

#include <stdio.h>

int main() {
    int num = 7;
    for (int i = 1; i <= 10; i++) {
        printf("%d * %d = %d\n", num, i, num * i);
    }
    return 0;
}

Example 3: Calculate Factorial of a Number

This example calculates the factorial of 5 (5! = 5 * 4 * 3 * 2 * 1) using a loop that counts down.

#include <stdio.h>

int main() {
    int n = 5;
    long long factorial = 1;
    for (int i = n; i >= 1; i--) {
        factorial *= i;
    }
    printf("Factorial of %d = %lld\n", n, factorial);
    return 0;
}

The `while` Loop

The `while` loop, an entry-controlled loop, repeatedly executes a block of code as long as a given condition is true. The condition is checked *before* the loop body is executed. It's used when the number of iterations is not known beforehand.

Syntax:

while (condition) {
    // code to be executed
    // update statement
}

Example 1: Sum of First 10 Natural Numbers

This program calculates the sum of numbers from 1 to 10 using a `while` loop.

#include <stdio.h>

int main() {
    int i = 1, sum = 0;
    while (i <= 10) {
        sum += i;
        i++;
    }
    printf("Sum = %d\n", sum);
    return 0;
}

Example 2: Reverse a Number

This code reverses the digits of an integer. The loop continues as long as the number is not zero.

#include <stdio.h>

int main() {
    int n = 12345, reversedN = 0, remainder;
    while (n != 0) {
        remainder = n % 10;
        reversedN = reversedN * 10 + remainder;
        n /= 10;
    }
    printf("Reversed Number = %d\n", reversedN);
    return 0;
}

Example 3: Count Digits in an Integer

This example counts the total number of digits in a given integer by repeatedly dividing it by 10.

#include <stdio.h>

int main() {
    long long n = 9876543210;
    int count = 0;
    while (n != 0) {
        n /= 10; // n = n / 10
        ++count;
    }
    printf("Number of digits: %d\n", count);
    return 0;
}

The `do-while` Loop

The `do-while` loop is an exit-controlled loop. It is similar to a `while` loop, but the loop body is executed at least once, regardless of whether the condition is true or false. The condition is checked *after* the loop body is executed.

Syntax:

do {
    // code to be executed
    // update statement
} while (condition);

Example 1: Print Numbers 1 to 5

Even if the condition was `i <= 0`, this loop would run once and print '1'.

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("%d ", i);
        i++;
    } while (i <= 5);
    printf("\n");
    return 0;
}

Example 2: Read Input Until a Positive Number is Entered

This is a perfect use case for `do-while`. The program must ask for input at least once. This simulation enters -2, then 5.

#include <stdio.h>

int main() {
    int number;
    do {
        printf("Enter a positive integer: ");
        scanf("%d", &number);
    } while (number <= 0);

    printf("You entered: %d\n", number);
    return 0;
}

Example 3: A Simple Menu System

A `do-while` loop ensures the menu is displayed at least once. The loop continues until the user chooses to exit (by entering 0).

#include <stdio.h>

int main() {
    int choice;
    do {
        printf("\n--- MENU ---\n");
        printf("1. Option 1\n");
        printf("2. Option 2\n");
        printf("0. Exit\n");
        printf("Enter your choice: ");
        scanf("%d", &choice);

        // This is where you would handle the choice
        printf("You chose: %d\n", choice);

    } while (choice != 0);

    printf("Exiting program.\n");
    return 0;
}
