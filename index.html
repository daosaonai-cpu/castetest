Understanding Arrays in C

In C programming, an array is a fundamental data structure used to store a collection of elements of the same data type in contiguous memory locations. Think of it as a set of numbered boxes, where each box holds a value, and you can access any box using its number, which is called an index. The index always starts from 0 for the first element. To declare an array, you must specify the type of data it will hold, a name for the array, and the number of elements it can store inside square brackets `[]`. For example, `int scores[10];` declares an integer array named `scores` that can hold 10 integer values. The indices for this array would range from 0 to 9. You can initialize an array at the time of declaration, like this: `int numbers[5] = {10, 20, 30, 40, 50};`. If you provide initial values, you can even omit the size, and the compiler will calculate it for you: `int numbers[] = {10, 20, 30};`. Accessing an element is simple: you use the array name followed by the index in square brackets. For instance, to get the third element from the `numbers` array, you would use `numbers[2]`, which would give you the value 30. Similarly, you can change the value of an element, for example: `numbers[2] = 35;`. It's crucial to remember that C does not perform bounds checking on arrays, meaning if you try to access an element outside the valid index range (e.g., `scores[10]` in our first example), it can lead to unpredictable behavior and bugs in your program.
