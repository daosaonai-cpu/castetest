C Array Tutorial

C Programming Array Tutorial

What is an Array?

An array is a collection of items of the same data type stored at contiguous memory locations.

Imagine you want to store the grades of 5 students. Instead of creating 5 different variables (like grade1, grade2, grade3...), you can use a single array variable called grades to store all 5 values.

Declaring an Array

To declare an array in C, you specify the data type of the elements and the number of elements the array will hold (its size).

Syntax: type arrayName[arraySize];

// This declares an array named 'scores' that can hold 10 integers.
int scores[10];

// This declares an array named 'averages' that can hold 5 floating-point numbers.
float averages[5];

// This declares an array named 'grades' that can hold 20 characters.
char grades[20];

Initializing an Array

You can initialize an array at the time of declaration by providing the values in curly braces {}. 

// Initialize an integer array with 5 values
int marks[5] = {90, 85, 92, 78, 95};

// You can omit the size, and the compiler will calculate it
int ages[] = {20, 22, 21, 19}; // Compiler knows size is 4

// If you provide fewer values than the size, the rest are set to 0
float salary[10] = {500.5, 650.0}; // First 2 are set, remaining 8 are 0.0

Accessing Array Elements

You can access individual elements of an array using an index.

Crucial Point: Zero-Based Indexing

In C, array indexing starts at 0, not 1.
- The first element is at index 0. - The second element is at index 1. - ... - For an array of size N, the last element is at index N-1. 

int marks[5] = {90, 85, 92, 78, 95};

// Access the first element (at index 0)
int firstMark = marks[0]; // firstMark is 90

// Access the third element (at index 2)
printf("The third mark is: %d\n", marks[2]); // Prints 92

// Change the value of the last element (at index 4)
marks[4] = 98; // The 5th element (95) is now 98

Arrays and Loops

Arrays and for loops are a perfect match. You can use a loop to easily iterate over every element in an array.

Example: Print all elements and calculate their sum. 

#include <stdio.h>

int main() {
    int ages[] = {20, 22, 21, 19, 23};
    int sum = 0;
    int size = 5;

    printf("Listing all ages:\n");
    
    // Loop from index 0 up to (but not including) 5
    for (int i = 0; i < size; i++) {
        // Print the element at the current index 'i'
        printf("Age at index %d: %d\n", i, ages[i]);
        
        // Add the current element to the sum
        sum = sum + ages[i];
    }

    printf("The sum of all ages is: %d\n", sum);
    
    return 0;
}

Common Pitfalls

  • Off-by-One Error: Forgetting that arrays start at index 0. For int arr[5], the valid indices are 0, 1, 2, 3, and 4.
  • Accessing Out of Bounds: Trying to access an element that doesn't exist, like arr[5] or arr[-1]. This is called a "buffer overflow" and can crash your program or lead to unpredictable behavior. C does not stop you from doing this!
  • Uninitialized Arrays: Using an array without initializing it first. The array will contain "garbage values" (whatever was in that memory location before).

A Quick Look: Multi-dimensional Arrays

You can also have arrays of arrays, known as multi-dimensional arrays. A 2D array is like a grid or a table. 

// Declare a 2D array (3 rows, 4 columns)
int matrix[3][4] = {
    { 1,  2,  3,  4 }, // Row 0
    { 5,  6,  7,  8 }, // Row 1
    { 9, 10, 11, 12 }  // Row 2
};

// Access the element at Row 1, Column 2
int element = matrix[1][2]; // element is 7

// You need nested loops to iterate over a 2D array
for (int i = 0; i < 3; i++) { // Loop over rows
    for (int j = 0; j < 4; j++) { // Loop over columns
        printf("%d ", matrix[i][j]);
    }
    printf("\n"); // Newline after each row
}
