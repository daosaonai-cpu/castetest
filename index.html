C Programming Loops: A Crash Course

Your guide to iteration and control flow in C.

What are Loops?

In programming, a loop is a control structure used to repeat a block of code as long as a specified condition is true. Loops are fundamental for automating repetitive tasks, processing collections of data like arrays, and implementing complex algorithms. Without them, you would have to write the same lines of code over and over again. C provides several types of loops to handle different situations.

The `for` Loop

The for loop is ideal when you know in advance how many times you want to execute a statement or a block of statements. It combines initialization, condition checking, and an increment/decrement operation in a single line.

Syntax:

for (initialization; condition; increment/decrement) {
   // code to be executed
}

Example:

This example prints numbers from 1 to 5.

#include <stdio.h>

int main() {
   int i;
   
   // Loop from 1 to 5
   for (i = 1; i <= 5; i++) {
      printf("%d\n", i);
   }
   
   return 0;
}

The `while` Loop

The while loop is used when the number of iterations is not known beforehand. It repeatedly executes a target statement as long as a given condition is true. The condition is checked *before* the loop body is executed.

Syntax:

while (condition) {
   // code to be executed
}

Example:

This example does the same thing as the `for` loop, but using a `while` loop structure.

#include <stdio.h>

int main() {
   int i = 1; // Initialization
   
   while (i <= 5) { // Condition
      printf("%d\n", i);
      i++; // Increment
   }
   
   return 0;
}

The `do...while` Loop

The do...while loop is similar to a while loop, but with a key difference: the loop body is guaranteed to execute at least once. This is because the condition is checked *after* the body of the loop has been executed.

Syntax:

do {
   // code to be executed
} while (condition);

Example:

This example asks for user input and continues to do so until the user enters a number greater than 10.

#include <stdio.h>

int main() {
   int number;
   
   do {
      printf("Enter a number greater than 10: ");
      scanf("%d", &number);
   } while (number <= 10);
   
   printf("Correct! You entered %d\n", number);
   
   return 0;
}

The `break` Statement

The break statement is used to exit a loop (for, while, or do...while) or a switch statement prematurely. When a break statement is encountered inside a loop, the loop is immediately terminated, and program control resumes at the next statement following the loop.

Example:

This loop intends to run 10 times, but it stops when i becomes 5.

#include <stdio.h>

int main() {
   int i;
   
   for (i = 1; i <= 10; i++) {
      if (i == 5) {
         printf("Breaking the loop at i = 5\n");
         break; // Exit the loop
      }
      printf("%d\n", i);
   }
   
   return 0;
}

The `continue` Statement

The continue statement is used to skip the rest of the current iteration of a loop and move to the next iteration. Unlike break, it doesn't terminate the loop entirely. It just bypasses the remaining code inside the loop for the current iteration.

Example:

This example prints numbers from 1 to 10 but skips printing the number 5.

#include <stdio.h>

int main() {
   int i;
   
   for (i = 1; i <= 10; i++) {
      if (i == 5) {
         printf("(Skipping 5...)\n");
         continue; // Skip this iteration
      }
      printf("%d\n", i);
   }
   
   return 0;
}
