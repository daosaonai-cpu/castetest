The Concept: Linear Regression

Linear regression is a foundational machine learning algorithm. It attempts to model the relationship between variables by fitting a linear equation to observed data. In this workbench, we act as a real estate agent trying to predict House Price (Y) based solely on the House Size (X).

The core equation being solved: Price = (Weight × Size) + Bias
