The Brain: Central Processing Unit (CPU)

The CPU is the primary processing unit of a computer. Think of it as the brain—nothing happens without its command.

  • Executes software instructions and processes complex data.
  • Controls and coordinates all computer operations.
  • Every single task performed by a computer passes through the CPU at lightning speed.

Main Components of the CPU

Arithmetic Logic Unit (ALU)

The math genius of the computer.

  • Performs all mathematical calculations (Add, Subtract, etc.).
  • Handles logical comparisons (Greater than, Equal to, AND/OR).

Control Unit (CU)

The traffic cop of the system.

  • Controls all computer operations and subsystems.
  • Directs the flow of data between hardware components.
  • Decodes instructions from memory.

Registers

The CPU's ultra-fast pocket memory.

  • Small, high-speed storage locations built directly into the CPU.
  • Temporarily hold data and instructions immediately needed for processing.

Memory & Storage: Primary Memory

Main memory is directly accessed by the CPU. It is extremely fast and essential for the smooth, real-time operation of your computer.

RAM

Random Access Memory

  • • Temporary (Volatile) workspace.
  • • Stores data and apps currently in use.
  • • Analogy: A whiteboard that gets wiped clean when the power turns off.

ROM

Read Only Memory

  • • Permanent (Non-volatile) memory.
  • • Stores essential boot-up instructions (firmware).
  • • Analogy: A printed book; you can read it, but you can't easily change the ink.

Secondary Memory (Storage)

Used for massive, long-term storage of your files, operating system, and installed programs.

  • Retains data when powered off.
  • Much larger capacity than RAM.
  • Considerably slower than Primary Memory.
  • Often measured in Gigabytes (GB) or Terabytes (TB).

Common Examples

HDD (Hard Disk) SSD (Solid State) Pen Drive Memory Card

Understanding Software

Hardware provides the physical foundation, but Software provides the instructions. Software is broadly categorized into 5 main types.

1. System
2. Application
3. Utility
4. Proprietary
5. Open Source

1. System Software

The foundational layer.

  • Manages hardware resources completely.
  • Acts as the core interface between the user, application software, and hardware.
  • Runs continuously in the background.

Examples

Microsoft Windows • macOS • Ubuntu Linux • Android

2. Application Software

The tool for the user.

  • Designed to perform specific tasks directly for end-users.
  • Helps complete day-to-day work, communication, or entertainment.
  • Runs "on top" of system software.

Examples

Microsoft Word • Google Chrome • Photoshop • Games

3. Utility Software

Software used specifically to maintain, protect, and optimize the computer system. They are the "mechanics" of your PC.

Protection

Defends against viruses and malware.
Ex: Antivirus

Optimization

Removes junk files and improves speeds.
Ex: Disk Cleanup

Maintenance

Safeguards data in case of failure.
Ex: Backups

Distribution Models

4. Proprietary Software

"Closed Source"

  • Owned by an individual or commercial company.
  • Source code is hidden and not publicly available.
  • Requires a paid license or subscription to use.
  • Cannot be legally modified or redistributed by users.
Examples:

Microsoft Office, Adobe Photoshop, macOS

5. Open Source Software

"Power to the People"

  • Source code is freely visible and available to anyone.
  • Usually free to download, use, modify, and share.
  • Developed collaboratively by global communities.
  • Promotes transparency and rapid innovation.
Examples:

Ubuntu, Mozilla Firefox, Android (AOSP)
