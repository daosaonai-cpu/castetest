int num; printf("Enter a number: "); scanf("%d", &num); // %d is a format specifier for int printf("You entered: %d", num);

7. Control Flow

  • if-else: Executes code based on a condition. 
    if (age >= 18) { ... } else { ... }
  • switch: A multi-way branch statement. 
    switch(choice) { case 1: ...; break; default: ...; }
  • for loop: Repeats a block of code a known number of times. 
    for (int i = 0; i < 5; i++) { ... }
  • while loop: Repeats as long as a condition is true. 
    while (condition) { ... }

8. Functions

A function is a block of code that performs a specific task. It helps in code reusability.

int add(int a, int b) { // Function definition
    return a + b;
}
int main() {
    int sum = add(5, 3); // Function call
    return 0;
}

9. Arrays & Strings

An array is a collection of items of the same data type stored at contiguous memory locations. Example: int numbers[5] = {10, 20, 30, 40, 50};. A string in C is an array of characters ending with the null character '\0'. Example: char name[] = "John";

10. Pointers

A pointer is a variable that stores the memory address of another variable. The & operator gives the address, and the * operator dereferences (gets the value at an address).

int var = 10;
int *p;      // Pointer declaration
p = &var;    // Store address of var in p
printf("Value: %d", *p); // Access value using pointer
