Introduction to Operators & Expressions

In programming, an operator is a symbol that tells the compiler to perform a specific mathematical or logical manipulation. Think of them as the action words of programming.

An expression is a combination of variables, constants, and operators that represents a single value. For example, 5 + 10 is an expression whose value is 15.

Arithmetic Operators

These are used to perform common mathematical calculations.

Operator Name Example Result
+ Addition 10 + 5 15
- Subtraction 10 - 5 5
* Multiplication 10 * 5 50
/ Division 10 / 5 2
% Modulus (Remainder) 10 % 3 1 
int a = 10;
int b = 3;

printf("Sum: %d\n", a + b);       // Output: 13
printf("Difference: %d\n", a - b); // Output: 7
printf("Product: %d\n", a * b);    // Output: 30
printf("Quotient: %d\n", a / b);   // Output: 3 (integer division)
printf("Remainder: %d\n", a % b);  // Output: 1

These operators compare two values and return a boolean result: true (1) or false (0).

int age = 20;

Logical Operators

Used to combine two or more conditions/constraints.

Operator Name Description
&&Logical ANDTrue only if ALL conditions are true.
||Logical ORTrue if AT LEAST ONE condition is true.
!Logical NOTReverses the state (true becomes false, false becomes true).
 
int hasLicense = 1; // 1 for true
int hasCar = 0;     // 0 for false

// AND example
if (hasLicense && hasCar) {
    printf("You can drive.\n");
}

// OR example
if (hasLicense || hasCar) {
    printf("You have either a license or a car (or both).\n");
}

// NOT example
if (!hasCar) {
    printf("You do not have a car.\n");
}

Increment & Decrement Operators

Used to increase (++) or decrease (--) a variable's value by one. They have two forms: prefix and postfix.

  • Prefix (++var): Increments the value before using it in the expression.
  • Postfix (var++): Increments the value after using its original value in the expression.
int a = 5;
int b;

// Postfix example
b = a++; // b gets the original value of a (5), then a becomes 6
printf("a = %d, b = %d\n", a, b); // Output: a = 6, b = 5

int x = 5;
int y;

// Prefix example
y = ++x; // x becomes 6, then y gets the new value of x (6)
printf("x = %d, y = %d\n", x, y); // Output: x = 6, y = 6

Conditional (Ternary) Operator

A shorthand for an if-else statement. It takes three operands.

condition ? value_if_true : value_if_false 
int score = 85;
char* result;

// Using if-else
if (score > 60) {
    result = "Pass";
} else {
    result = "Fail";
}
printf("Result: %s\n", result);

// Using conditional operator
char* result_ternary = (score > 60) ? "Pass" : "Fail";
printf("Result (Ternary): %s\n", result_ternary);

Bitwise Operators

These operators perform operations on individual bits (0s and 1s) of integer types. They are used in low-level programming.

Operator Name Description
&Bitwise ANDSets bit to 1 if both bits are 1.
|Bitwise ORSets bit to 1 if at least one bit is 1.
^Bitwise XORSets bit to 1 if the bits are different.
~Bitwise NOT (Complement)Flips all the bits.
<<Left ShiftShifts bits to the left, fills with 0s.
>>Right ShiftShifts bits to the right.
 
unsigned int a = 60; // 0011 1100
unsigned int b = 13; // 0000 1101
int result = 0;

result = a & b;      // 12 -> 0000 1100
result = a | b;      // 61 -> 0011 1101
result = a ^ b;      // 49 -> 0011 0001
result = ~a;         // -61 -> 1100 0011 (in 2's complement)
result = a << 2;     // 240 -> 1111 0000
result = a >> 2;     // 15  -> 0000 1111

Operator Precedence & Associativity

Precedence determines which operator is performed first in an expression with multiple operators. For example, multiplication has higher precedence than addition.

Associativity determines the order in which operators with the same precedence are evaluated. It can be Left-to-Right or Right-to-Left.

Precedence Operators Associativity
Highest() [] -> . ++ -- (postfix)Left-to-right
...++ -- (prefix) ! ~ * & sizeofRight-to-left
...* / %Left-to-right
...+ -Left-to-right
...<< >>Left-to-right
...< <= > >=Left-to-right
...== !=Left-to-right
...& ^ | && ||Left-to-right
...?:Right-to-left
...= += -= *= /= %=Right-to-left
Lowest,Left-to-right
 
// Precedence example
int result = 5 + 2 * 3; // 2 * 3 is done first (11)
printf("Result: %d\n", result);

// Associativity example
int a = 10, b = 5, c = 2;
result = a / b * c; // a / b is done first, then * c (4)
printf("Result: %d\n", result);

// Use parentheses to override precedence
result = (5 + 2) * 3; // 5 + 2 is done first (21)
printf("Result: %d\n", result);
